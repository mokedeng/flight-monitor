# Flight Price Monitor (MCP Version)

基于 Microsoft Playwright MCP 的机票价格监控和预测工具。

## 功能特点

- 🔍 **多平台抓取**: 支持去哪儿、携程、飞猪、智行火车票
- ✈️ **官网查询**: 优先查询航空公司官网（山东航空、深圳航空、南方航空等）
- ⏰ **自动监控**: 每1小时自动检查价格变化
- 📊 **价格预测**: 基于历史数据和季节因素预测未来30天价格
- 🔔 **智能提醒**: 价格达到目标时自动通知
- 💾 **本地存储**: 所有数据存储在本地，保护隐私
- 📋 **对比表格**: 同一航班各平台价格一键对比

### 🆕 最新增强功能

- 🗓️ **智能日期解析**: 支持自然语言日期表达
  - 模糊日期："下周五"、"本周末"、"清明节"
  - **组合表达式**："清明节前的上个周六"、"国庆后3天"
  - 相对日期："下周六"、"下下周一"

- 🎭 **节假日缓冲期预警**: 智能检测节前节后价格波动
  - 节前2天预热期（系数1.3x）
  - 节前1天出行高峰（系数1.5x）
  - 节后1天返程余温（系数1.2x）

- 🛫 **多机场精准定位**: 支持具体机场名称
  - 多机场城市：北京（首都/大兴）、上海（虹桥/浦东）、成都（双流/天府）
  - 直接指定："从大兴出发"、"到浦东"、"天府机场"

- 💰 **增强价格解析**: 智能识别多种价格格式
  - 支持 k/K/千 单位（1.2k → 1200元）
  - 智能过滤无效价格（<100 或 >20000）

- ⚡ **并发性能优化**: 节假日API请求提速4倍
  - 使用 Promise.all 并发查询
  - 缓存失效时仍保持快速响应

## 价格预测算法

系统使用以下因素预测价格：

1. **季节因素**: 不同月份的价格系数
2. **提前购买**: 出发前天数的影响
3. **历史趋势**: 基于历史数据的趋势分析
4. **星期效应**: 周末vs工作日的价格差异
5. **节假日缓冲**: 节前节后价格波动预警

## 配置选项

```bash
# 自定义数据目录
export FLIGHT_MONITOR_DATA_DIR="$HOME/.flight-monitor"

# 通知 Webhook
export FLIGHT_MONITOR_ALERT_WEBHOOK="https://your-webhook-url"
```

## 注意事项

- 建议提前 30-60 天开始监控
- 周二、周三通常价格较低
- 避开节假日和展会期间
- 直飞航班价格通常比中转高
- 智行火车票可能因地区限制无法访问
- 去哪儿等平台可能需要登录

## 技术架构

### 核心依赖

- **Microsoft Playwright MCP**: 浏览器自动化，用于抓取实时价格
- **Timor.tech API**: 中国节假日查询服务（含调休逻辑）

### 性能优化

- 并发节假日API请求（4倍性能提升）
- 智能等待策略（避免固定延迟）
- 快速失败机制（防止触发WAF）

### 反爬虫策略

- 使用移动端URL
- 模拟真实用户行为
- 多平台轮换访问
- 设置合理超时时间
